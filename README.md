# Router_verification_UVM
Developed a complete UVM-based verification environment for packet router design verification using SystemVerilog. Implemented reusable UVM components including driver, monitor, sequencer, agent, scoreboard, and functional coverage. Verified packet routing functionality, protocol compliance, error handling, and data integrity through constrained-random and directed test cases. Achieved coverage-driven verification and transaction-level modeling (TLM).

Features
UVM-based reusable verification environment
Driver, Monitor, Sequencer, Agent, Scoreboard
Constrained Random Verification
Functional Coverage 
Packet Routing Verification
Transaction-Level Modeling (TLM)
Protocol and Data Integrity Checking
